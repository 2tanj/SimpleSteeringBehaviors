# Simple Steering Behaviors
AI Movement in Unity containing the following steering behaviours: 
 - `Arrive`
 - `Evade`
 - `Flee`
 - `Path Follow`
 - `Pursuit`
 - `Seek`
 - `Wander`

# Examples 
All steering behaviours are demonstrated in their own scenes found in the `_Scenes` folder
